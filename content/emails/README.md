## File Naming Convention

```
YYYY-MM-DD_recipient_subject.md

Examples:
2026-02-12_john.doe@example.com_project_update.md
2026-02-12_team@company.com_meeting_agenda.md
2026-02-12_info@learningplatform.com_course_recommendation.md
```

## Metadata Standard

Each email content file should include:

```markdown
---
title: "Email Subject Line"
created: "2026-02-12"
generated_by: "Email Manager Skill"
sender: "claude@daily-life-fte.com"
recipient: "john.doe@example.com"
subject: "Project Update - Week 6"
status: "draft/sent"
priority: "high/normal/low"
category: "professional/personal/meeting/follow-up"
related_workflow: "morning-routine/work-preparation"
---

# Email Body
```

5. **Archive**: Sent email saved to directory
6. **Follow-up**: Schedule any necessary follow-ups
