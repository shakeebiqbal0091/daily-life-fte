# Social Media Content Directory

This directory stores all generated social media content from the Daily Life Productivity Digital FTE system.

## Purpose

Generated social media posts for:
- LinkedIn (professional content)
- Twitter (engaging threads and updates)
- Instagram (visual storytelling and captions)

## Content Types

### LinkedIn Posts
Professional content including:
- Career advice and insights
- Learning achievements and milestones
- Industry observations and trends
- Professional development tips

### Twitter Threads
Engaging content including:
- Industry analysis and commentary
- Quick tips and actionable advice
- Learning resources and recommendations
- Professional networking insights

### Instagram Captions
Visual storytelling including:
- Learning journey documentation
- Productivity tips and hacks
- Wellness and lifestyle content
- Professional achievements

## File Naming Convention

```
YYYY-MM-DD_platform_topic.md

Examples:
2026-02-12_linkedin_career_advice.md
2026-02-12_twitter_productivity_tips.md
2026-02-12_instagram_learning_journey.md
```

## Metadata Standard

Each social media content file should include:

```markdown
---
title: "Content Title"
created: "2026-02-12"
generated_by: "Content Creator Agent"
platform: "linkedin/twitter/instagram"
topic: "career_advice/productivity_tips/learning_journey"
status: "draft/final"
scheduled_date: "2026-02-12"
hashtags: ["#productivity", "#learning", "#career"]
---

# Content Body
```

## Content Generation

Content is generated by the Content Creator Agent using:
- Current learning and achievements

## Review Process

1. **Generation**: Agent creates content with metadata
2. **Review**: User reviews content for accuracy and tone
3. **Approval**: User approves or requests edits
4. **Scheduling**: Content scheduled for optimal timing
5. **Posting**: Content posted to appropriate platform
6. **Engagement**: Monitor and respond to engagement
